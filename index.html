Tema 1

Propiedad Valor Inicial Descripción
color currentColor Establece el color del texto.
font-family serif (depende del sistema) Especifica la fuente del texto, pudiendo ser una fuente específica o una lista de fuentes.
font-size medium (normalmente 16px) Define el tamaño de la fuente.
font-weight normal (400) Establece el grosor del texto (e.g., normal, bold).
font-style normal Define el estilo de la fuente (e.g., normal, italic, oblique).
font-variant normal Controla el uso de letras en minúsculas pequeñas (small caps) y otras variantes.
font inherit Propiedad abreviada para establecer varias propiedades de fuente a la vez.
text-align start (depende del idioma) Alinea el texto dentro de un contenedor (e.g., left, right, center, justify).
line-height normal (1.2 veces el tamaño de fuente) Establece el espacio entre líneas de texto.
text-decoration none Añade decoraciones al texto, como subrayado o tachado.
text-transform none Controla la capitalización del texto (e.g., uppercase, lowercase).
vertical-align baseline Alinea elementos en línea o celdas de tabla verticalmente.
text-indent 0 Indenta la primera línea de un bloque de texto.
letter-spacing normal Ajusta el espacio entre caracteres.
word-spacing normal Ajusta el espacio entre palabras.
white-space normal Controla cómo se maneja el espacio en blanco dentro de un elemento.

Tema 2

Característica Propiedades Descripción
Tamaño, Color y Decoración para el tamaño podemos usar width, height, font-size, max-width, max-height, min-width, min-height, box-sizing, para el color usamos las propiedades color, background-color, border-color, outline-color y para finalizar en el caso de la decoracion usamos text-decoration, text-shadow, border, border-radius, box-shadow, opacity, background-image y filter. Las propiedades de tamaño, color y decoración en CSS son esenciales para crear un diseño web atractivo y funcional. Las propiedades de tamaño, como width y height, permiten definir las dimensiones de los elementos, asegurando que se adapten a diferentes dispositivos y resoluciones. Por otro lado, las propiedades de color, incluyendo color y background-color, mejoran la estética y la legibilidad al permitir personalizar los colores del texto y del fondo. Finalmente, las propiedades de decoración, como border, box-shadow y text-decoration, añaden efectos visuales que enriquecen la presentación, brindando profundidad y estilo a los elementos. Al combinarlas adecuadamente, se puede lograr un diseño cohesivo y visualmente atractivo que mejora la experiencia del usuario.
Decoración Personalizada border-radius, box-shadow, background-image, background-size, text-shadow y @keyframes

La decoración personalizada en CSS es una poderosa herramienta que permite a los diseñadores web crear interfaces visualmente impactantes y únicas. Al combinar diversas propiedades como bordes, colores, sombras y fondos, se pueden transformar elementos comunes en componentes atractivos que capturan la atención del usuario. Por ejemplo, el uso de border-radius para redondear esquinas, junto con box-shadow para añadir profundidad, puede hacer que un botón o un cuadro de contenido se vea más moderno y amigable.

Además, la aplicación de imágenes de fondo y gradientes puede enriquecer la estética de un diseño. Utilizando propiedades como background-image y background-size, los desarrolladores pueden crear fondos dinámicos que complementan el contenido. Los gradientes, en particular, ofrecen una forma fácil de agregar color y dimensión sin la necesidad de imágenes externas, lo que mejora la velocidad de carga de la página y mantiene el diseño ligero.

Finalmente, las animaciones y efectos de texto son cruciales para dar vida a los elementos en una página web. Propiedades como text-shadow y animaciones con @keyframes pueden hacer que el texto destaque y que los elementos reaccionen de manera interactiva al pasar el ratón. Al integrar estas técnicas, los desarrolladores no solo mejoran la apariencia visual de sus sitios, sino que también fomentan una experiencia de usuario más rica y atractiva, esencial para mantener la atención y el interés del visitante.

Tema 3

Característica Propiedades Descripción
Anchura y Altura width, height, vw, vh, box-sizing, box-sizing: border-box

La propiedad width establece el ancho de un elemento, mientras que height controla su altura. Ambas propiedades pueden recibir valores en diferentes unidades, como píxeles (px), porcentajes (%), em, rem, entre otros, lo que proporciona flexibilidad en el diseño.

Cuando se trabaja con diseño responsivo, la utilización de porcentajes y unidades relativas como vw (viewport width) y vh (viewport height) se vuelve crucial. Esto permite que los elementos se ajusten automáticamente según el tamaño de la pantalla del dispositivo. Por ejemplo, un contenedor con un ancho del 50% ocupará la mitad de la pantalla, independientemente de su tamaño. Esta capacidad de adaptación es esencial para crear experiencias de usuario fluidas en dispositivos móviles y de escritorio.

Además, las propiedades de anchura y altura pueden interactuar con otras características de diseño, como el box-sizing, que influye en cómo se calculan estos valores. Con box-sizing: border-box, el ancho y la altura de un elemento incluyen padding y bordes, lo que simplifica el control del espacio ocupado. El uso de estas propiedades, junto con técnicas como flexbox o grid, permite a los diseñadores crear interfaces atractivas y bien estructuradas, optimizando la presentación de contenidos y la usabilidad en general.
Eliminar el Borde border: none se utiliza para eliminar cualquier borde de un elemento.
